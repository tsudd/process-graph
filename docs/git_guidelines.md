# Git usage reference

## Commit messages
Use the following format for commit messages:
```markdown
upd: Short description of the change (max 50 characters)
feat: Short description of the new feature (max 50 characters)
fix: Short description of the bug fix (max 50 characters)
rm: Short description of the removed code (max 50 characters)
test: Short description of the test changes (max 50 characters)

```
